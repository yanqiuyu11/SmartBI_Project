# SmartBI_Project
风险评估系统报表展示项目
# 说明
展示风险评估系统`报表`UI`优化`前后效果
## 优化前
主要使用`excel电子表格`和`excel图形`完成的报表，用来展示新增贷款、新增客户数以及新增放款笔数等指标
